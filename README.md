- **Remove Obstructive Overlays:** Popups, cookie banners, and adblock detection walls.

### Power Features
- **Enable Reader Mode:** Focus on the main content without distractions.

- **Stop Animations:** Freeze distracting animations or auto-playing elements.

### Automation & Settings
- **Master Automation:** Clean every page automatically as you browse.  
- **Automation Delay:** Control how many seconds the extension waits before cleaning (Recommended: 2s).  
- **Custom Site Rules:** Apply different cleaning levels to specific domains (e.g., only dark mode on Wikipedia).  
- **Global Exclusions:** List domains to be ignored entirely by the extension.

## ðŸ“– Frequently Asked Questions

**What is "Fix This Page"?**  
A web purifier that removes intrusive elements, sticky headers, iframes, and clutter, making pages more readable and usable.

**How do I use Master Automation?**  
Enable the switch to clean every page automatically. Adjust the Automation Delay to ensure the page fully loads before cleanup.

**What are Site Rules?**  
Site Rules let you apply different cleaning levels to individual domains, like enabling only dark mode on Wikipedia but full cleanup on news sites.

**When should I use "Remove Scripts"?**  
Only use this on very cluttered sites. It is aggressive and can break buttons or interactive menus.

**What is Debug Mode?**  
Debug Mode logs additional information in the console for troubleshooting. Open DevTools with F12 (Ctrl+Shift+I / Cmd+Option+I on Mac).
